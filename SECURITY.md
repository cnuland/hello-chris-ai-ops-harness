# Security Policy

## Supported Platforms
- OpenShift 4.21.x target (manifests and docs are authored against this baseline).

## Reporting a Vulnerability
Please open a private disclosure (e.g., via security@your-org.example) with:
- Affected components and versions
- Reproduction steps and impact
- Any suggested mitigations

We will acknowledge within 5 business days and provide an expected remediation timeline.
